3 Indicateurs de vegetation

Le satellite mesure la lumiere reflechie par le sol dans differentes longueurs d'onde (visible, infrarouge). A partir de ces mesures, on calcule des indicateurs qui decrivent l'etat de la vegetation :

Indicateur Valeur moyenne A quoi ca sert Type
Verdure (NDVI) {{ "%.3f" | format(metadata.get('statistics', {}).get('NDVI', 0)) }} Mesure l'intensite du vert — distingue vegetation, sol nu, urbain Calcule
Vitalite (EVI) {{ "%.3f" | format(metadata.get('statistics', {}).get('EVI', 0)) }} Mesure la vigueur des plantes — distingue cocotiers de foret dense Calcule
Detection d'eau (NDWI) utilise en classification Identifie ocean, lacs, rivieres — exclut ces zones de l'analyse Calcule
Contraste infrarouge (NIR/R) utilise en classification Ratio complementaire pour affiner la detection des types de vegetation Calcule

4 Seuils de classification

Seuils definis pour distinguer les types de couverture du sol a partir des indicateurs de vegetation. Filtre de lissage : 30m.

Seuil Valeur Signification Type
Detection eau (exclue) {{ thresholds.eau_ndwi }} Au-dessus → classe comme eau (ocean, lacs) Estime
Verdure min sol nu (exclu) {{ thresholds.sol_nu_ndvi }} En dessous → sol nu ou ocean (pas de vegetation) Estime
Verdure min urbain {{ thresholds.urbain_ndvi }} Verdure entre {{ thresholds.urbain_ndvi }} et {{ thresholds.sol_nu_ndvi }} → zones urbanisees Estime
Verdure min cultures {{ thresholds.culture_ndvi_min }} Borne basse de verdure pour les cultures (canne, maraichage) Estime
Verdure max cultures {{ thresholds.culture_ndvi_max }} Borne haute de verdure pour les cultures Estime
Verdure min cocotiers {{ thresholds.cocotier_ndvi_min }} Borne basse de verdure pour les cocotiers Estime
Vitalite min cocotiers {{ thresholds.cocotier_evi_min }} Borne basse de vitalite pour les cocotiers Estime
Vitalite max cocotiers {{ thresholds.cocotier_evi_max }} Borne haute de vitalite (au-dessus = foret dense) Estime
Verdure min foret {{ thresholds.foret_ndvi }} Au-dessus + forte vitalite → foret dense Estime
Vitalite min foret {{ thresholds.foret_evi }} Au-dessus + forte verdure → foret dense Estime
